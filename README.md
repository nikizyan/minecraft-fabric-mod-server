# Termux Mobile Minecraft Server (Fabric + Playit.gg)

A complete guide and automation toolkit for hosting a modded Minecraft (Fabric) server directly on an Android device from scratch using Termux. This project features automated process management, system sleep prevention, and persistent public tunneling without port forwarding.

## 🚀 Features
* **Android-Native Hosting:** Run a full Java-based Fabric server using OpenJDK 21 on standard mobile hardware.
* **Zero-Port-Forwarding Tunnel:** Leverages `playit.gg` to maintain a static, persistent public IP/domain across network changes.
* **One-Command Automation:** Custom shell aliases handle wake locks, background process initialization, and safe shutdowns.
* **Optimized Architecture:** Configured completely inside isolated hidden directories (`~/.mc-server`) for a clean environment.
